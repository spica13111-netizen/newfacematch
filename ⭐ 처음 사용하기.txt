    📱 상품 매칭 프로그램 - 5분 만에 시작하기
═══════════════════════════════════════════════════════════════


🎯 이 프로그램이 뭐하는 건가요?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

엑셀 파일에 있는 상품 이름과 구글 시트에 있는 상품 이름을
자동으로 찾아서 매칭해주는 프로그램입니다.

예시:
엑셀: "나이키 에어맥스 270"
구글시트: "나이키 에어 맥스270"
→ 자동으로 찾아서 연결해줍니다!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🚀 방법 1: 웹으로 열기 (다른 사람도 함께 사용)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[1단계] 시작하기
    👉 "🌐 웹으로 시작.bat" 파일을 더블클릭하세요

    그러면 2개의 검은 창이 나타납니다:
    - 첫 번째 창: Streamlit 앱 (그냥 놔두세요)
    - 두 번째 창: ngrok 터널 (여기서 주소 확인!)

[2단계] 주소 복사하기
    👉 두 번째 창(ngrok)에서 다음과 같은 주소를 찾으세요:

    Forwarding  https://abc-def-ghi.ngrok-free.app -> ...
                ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                이 부분을 복사하세요!

[3단계] 접속하기
    👉 복사한 주소를 인터넷 주소창에 붙여넣기
    👉 "Visit Site" 버튼이 나오면 클릭
    👉 프로그램 화면이 나타납니다!

[4단계] 다른 사람과 공유
    👉 복사한 주소를 카톡/문자로 보내면
    👉 다른 사람도 똑같이 접속할 수 있어요!

[종료하기]
    👉 열린 검은 창 2개를 모두 닫으면 끝!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💻 방법 2: 내 컴퓨터에서만 사용
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[1단계] 시작하기
    👉 "💻 로컬 실행.bat" 파일을 더블클릭하세요

    검은 창이 하나 나타나고 자동으로 인터넷 창이 열립니다!

[2단계] 사용하기
    👉 자동으로 열린 인터넷 창에서 바로 사용하세요

[종료하기]
    👉 검은 창을 닫으면 끝!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 프로그램 사용법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

프로그램이 열리면:

[1단계] 엑셀 파일 선택
    👉 "📁 엑셀 파일 선택" 버튼 클릭
    👉 상품 목록이 있는 엑셀 파일 선택
    👉 잠시 기다리면 로딩 완료!

[2단계] 자동 매칭
    👉 "🤖 자동 매칭 시작" 버튼 클릭
    👉 프로그램이 자동으로 비슷한 상품 찾아서 연결해줍니다
    👉 몇 초만 기다리세요!

[3단계] 결과 확인
    👉 "🔍 매칭 결과 확인" 탭 클릭
    👉 어떤 상품끼리 연결됐는지 볼 수 있어요
    👉 잘못 연결된 건 직접 수정할 수 있어요!

[완료!]
    👉 구글 시트에 자동으로 저장됩니다!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❓ 자주 묻는 질문
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 검은 창이 안 나와요 / 에러가 나요
A. 💊 패키지 설치.bat 파일을 먼저 실행해주세요
   (처음 한 번만 하면 됩니다)

Q. 웹 주소가 안 열려요
A. 컴퓨터가 인터넷에 연결되어 있는지 확인하세요
   방화벽이 막고 있을 수 있으니 "허용" 눌러주세요

Q. 구글 시트에 저장이 안 돼요
A. config 폴더 안에 "Google Sheets API.json" 파일이
   있는지 확인하세요 (관리자에게 문의)

Q. 웹으로 열었는데 다른 사람이 접속 안 돼요
A. 주소를 정확히 복사했는지 확인하세요
   컴퓨터가 꺼지면 접속이 안 됩니다!

Q. 프로그램을 끄고 싶어요
A. 열린 검은 창들을 모두 닫으면 됩니다
   또는 "배포파일/❌ 웹 종료.bat" 실행

   • 웹으로 열었으면 컴퓨터를 켜두세요
   • 엑셀 파일은 닫고 업로드하세요

❌ 하면 안 되는 것:
   • 프로그램 사용 중에 검은 창 닫기
   • config 폴더 안의 파일 삭제/수정
   • 프로그램 폴더 이름 바꾸기
